# User Manual

## Browser-Bedienung

1. Startseite öffnen
2. Neues Benutzerkonto registrieren
3. Als Rider oder Provider anmelden
4. Provider: Scooter anlegen, aktualisieren oder löschen
5. Rider: verfügbaren Scooter ausleihen
6. Rider: aktive Ausleihe mit Enddaten beenden
7. Dashboard zeigt Token und letzte Ausleihen an

## Demo-Benutzer

- Provider: `provider1` / `Provider123!`
- Rider: `rider1` / `Rider123!`
